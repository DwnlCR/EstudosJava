Exercicio-métodos

Crie uma classe Funcionario com os seguintes atributos:

  • String nome;
  • int idade;
  • double [] salario; // tres salarios devem ser guardados

Crie dois métodos

  • Para imprimir os dados
  • Para tirar a media dos salarios e imprimir o resultado

Apos isso faça com que o metodo que imprime os dados chame o metodo que faz a media salarial


Tente rodar o codigo chamando o metodo imprime mas sem usar a inicialização do atributo funcionario.salarios


Inicialização padrão de cada um dos atributos pois se eles forem chamados sem valor algum eles terão seu valor inicial padrão

  • String nome = null;
  • int idade = 0;
  • double [] salario = null;

Agora vamos fazer uma condição para caso funcionario.salarios == null (inicialização padrão) dar um break: return; para parar o metodo que calcula a media salarial
